Spring MVC study project.

### Components
 * Spring Boot
 * Spring MVC
 * Thymeleaf
 * Spring Data JPA
 * Spock
 * Gradle

### Run application
You need to set a proper configuration in **application.properties** for a DB.

Then run:
```
$ ./gradlew bootRun
```
